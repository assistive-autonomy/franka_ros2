# Hardware Interface
## Overview

This hardware_interface package is borrowed from the [ros2_control repository](https://github.com/ros-controls/ros2_control) with custom modifications to meet our specific requirements.

## Modifications

We have added the **passing the manage overrun feature** on top of [commit ab304b5
](https://github.com/ros-controls/ros2_control/commit/ab304b5ca56eed61996e50a72808245958d46004) from the ros2_control repository.
